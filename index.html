java继承

发表于 2021-06-03

继承（程序中的继承）

  • 假设动物类成员属性（品种，名字，月份）等为公共父类。
  • 同样方法也可以设置共性（吃、跑、）

继承（单一继承，只能继承一个父类）

  • 一种类与类之间的关系
  • 使用已存在的类（父类或基类）定义作为基础建立新类（子类或派生类）
  • 新类的定义中可以增加新的数据和新的功能，也可以用父类的功能，但不能选择性的继承父类

继承的关系

  • 满足”A is a B”的关系就可以形成继承的类（猫和狗继承自动物类）

继承的实现

  • extends关键词实现

    1. 编写父类 class Animal{ 公共的属性和方法 }
    2. 编写自雷，继承父类
      class Dog extends Animal{ 子类特有的属性和方法 }

  • 单继承（一个子类只有唯一的父类）

  • 子类可以访问父类的非私有成员，父类不可以访问子类特有成员。

方法重写

  • 在子类中定义
  • 方法名（与父类一致）
  • 参数类型，顺序，个数（与父类完全一致）
  • 注意：当子类重写父类方法后，子类对象调用的是重写后的方法。

访问修饰符

  • 公有的：public
  • 私有的：private
  • 受保护的：protected
  • 默认

super关键词

super ：父类对象的引用

  • 代表父类应用
  1. 访问父类成员方法 super.print();
  2. 访问父类属性 super.name;
  3. 访问父类构造方法 super();
  • 子类的构造的过程中必须调用其父类的构造方法
  • 如果子类的构造方法中没有显式标注，则系统默认调用父类的无参构造方法
  • 如果子类构造方法中既没有显式标注，且父类中没有无参的构造方法，则编译出错
  • 使用super调用父类指定构造方法，必须在子类的构造方法的第一行

super和this（构造方法调用时，this和super不能同时使用）

  • this
    1. 访问当前类的成员方法
    2. 访问当前类的成员属性
    3. 访问当前类的构造方法
    4. 不能在静态方法中使用
  • super
  1. 访问父类的成员方法
  2. 访问父类的成员属性
  3. 访问父类的构造方法
  4. 不能在静态方法中使用

继承后的初始化顺序

父类静态成员–子类静态成员–父类对象构造–子类对象构造

方法重写

  • 在满足继承关系的子类中
  • 方法名、参数列表与父类相同
  • 返回值小于等于父类（即可以是子类）
  • 访问修饰符的限定范围大于等于父类方法

方法重载

  • 在同一个类中
  • 方法名相同
  • 参数个数、书序、类型不同
  • 返回值类型、访问修饰符任意

Object类

  • Object类是所有类的父类
  • 一个类没有使用extends关键字明确表示继承关系，则默认继承Object类（包括数组）
  • java中的每个类都可以使用Object中定义的方法

final（不能用来修饰构造方法）

  • final Class:该类没有子类
  • final 方法：该方法不允许被子类重写，但可以正常被子类继承使用
  • final 修饰方法中的局部变量时，只要在具体被使用之前进行赋值即可，一旦赋值不允许被修改
  • final 类中成员属性:1、定义时候初始化 2、在构造代码块或构造方法中能进行初始化操作

面向对象

发表于 2021-06-03

面向对象

  • 关于面向对象编程
  • 1、创建类；2、实例化对象；3、完成具体的程序

什么是对象（万物皆对象）

什么是面向对象

  • 人关注对象
  • 人关注事物信息

    什么是类

  • 类是模子，确定对象会拥有的特性（属性）和行为（方法）
  • 对象是类的实例表现
  • 类是对象的类型
  • 对象是特定类型的数据

属性和方法

  • 属性：对象具有各种静态特征
  • “对象有什么”
  • 方法：对象具有的各种动态行为

类和对象的关系

  • 类（抽象的概念，模板）
  • 对象（一个看的到、摸得到的实体）

对象实例化

  • 实例化对象的过程可以分为两个部分:
  • 声明对象 object_cat one
  • 实例化对象 new object_cat();

注意事项

  • 需要多次访问同一对象时，必须进行声明
  • 同一作用范围内，吧態定义同名对象
  • 可以同时声明多个引用，用逗号隔开

构造方法（构造函数）

  • 不能被对象单独调用
  • 构造方法与类同名且没有返回值
  • 只能在对象实例化的时候调用
  1. 构造方法的语句格式

总结

  • 面向对象（关注现实存在的事物各方面的信息，从对象得到角度出发，格局事物的特征进行程序设计）
  • 对象：用来描述客观事物的一个实体
  • 类：具有相同属性和方法的一组对象集合

封装

  • 修改属性的可见性（设为private 隐藏）
  • 创建getter/setter方法（设为public用于属性的读写）
  • 在getter/setter方法中加入属性控制语句（对属性的合法性进行判断）

如何跨包调用类

  1. 加载包下所有类（import）
  2. 加载包下指定类
  3. 直接加载 包.类

Java 常量与变量

标识符

  • 标识符可以由字母、数字、下划线和美元符组成，不能由数字开头
  • 标识符严格区分大小写

  • 标识符的命名最好能反应出起作用

变量

  • 什么是变量:变量是程序中数据临时存储
  • 变量的三元素: 变量类型、变量名、变量值

变量的命名规则

  • 满足便是符命名规则
  • 符合驼峰法命名规则
  • 尽量简单、做到见名知意
  • 变量名的长度没有限制

类的命名规则

  • 满足Pascal命名法规范（组成类名的命名首字母必须大写）

  • 基本数据类型

  • 字符型字面值用单引号内的单个字符表示。(‘a’、’b’、’&’)

  • 如果A类型的数据表示范围比B类型大，则将A类型的值赋值给B类型，需要强制类型转换
  • 强制数据类型转换格式:(数据类型)数值

表连接

内连接是常见的一种表连接，用于查询多张关系表符合连接条件的记录;

外连接：除了符合条件的记录之外，结果集中还会保留不符合条件的记录
左连接与右连接：左外连接就是保留左表所有的记录,与右表做连接。如果右表有符合条件的记录就与左表连接。如果右表没有符合条件的记录,就用 NULL 与左表连接。右外连接也是如此。

UNION 关键字可以将多个查询语句的结果集进行合并 要求：字段个数相同，字段名称相同

子查询

子查询是一种查询中嵌套查询的语句
子查询可以写在三个地方：WHERE子句、FROM子句、SELECT子句。
WHERE 子查询最简单，最容易理解，但是效率是最低的

FROM 子查询只会执行一次，效率很高

SELECT 子查询:每输出一条记录就要执行一次，查询效率很低

单行子查询和多行子查询
WHERE子句中，可以使用IN,ALL,ANY,EXISTS关键字来处理多行表达式结果集的条件判断
