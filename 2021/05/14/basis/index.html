Java 变量与常量

发表于 2021-05-14

Java 常量与变量

标识符

  • 标识符可以由字母、数字、下划线和美元符组成，不能由数字开头
  • 标识符严格区分大小写
  • 标识符不能是java关键词和保留字
  • 标识符的命名最好能反应出起作用

变量

  • 什么是变量:变量是程序中数据临时存储
  • 变量的三元素: 变量类型、变量名、变量值

变量的命名规则

  • 满足便是符命名规则
  • 符合驼峰法命名规则
  • 尽量简单、做到见名知意
  • 变量名的长度没有限制

类的命名规则

  • 满足Pascal命名法规范（组成类名的命名首字母必须大写）

数据类型

  • 基本数据类型
    • 数值型(int、long float、double)
    • 字符型(char)
    • 布尔型(boolean)
  • 引用数据类型
    • 类(class)
    • 接口(interface)
    • 数组

基本数据类型

数据类型 说明 字节
byte 字节型 1
short 短整型 2
int 整型 4
long 长整型 8
float 单精度浮点型 4
double 双精度浮点型 8
char 字符型 2
boolean 布尔型 1

变量声明

  • 格式: 数据类型 变量名;

赋值

  • 使用“=”运算符进行赋值
  • “=” 叫做赋值运算符，将运算符右边的值副歌左边的变量
  • 可以在定义变量的同时给变量赋值，即变量的初始化(int n=3;)

基本数据类型变量的存储

  • 按照作用范围分为：类级、对象实例级、方法级、块级

字面值(常量)

  • 字符型字面值用单引号内的单个字符表示。(‘a’、’b’、’&’)
  • 定义字符型变量 (char a= ‘a’;)
  • char ch = 65; (输出ASCII码 A )
  • 布尔类型的字面值只能定义成true和false
  • 字符串不属于基本数据类型是对象(双引号引起来的0或多个字符)

ASCII编码与Unicode编码

  • ASCII码是美国标准的信息交换码
  • ASCII码不能支持所有语言
  • Unicode编码又称为万国码
  • Unicode编码的目标是支持世界上所有的字符集

类型转换

  • 分为自动类型转换和强制类型转换
  • 自动(long n = 263;)
  • 强制(char ch = (char)65536;)可能引起数据丢失
  • 如果A类型的数据表示范围比B类型大，则将A类型的值赋值给B类型，需要强制类型转换
  • 强制数据类型转换格式:(数据类型)数值

常量

  • 命名规则：final (类型) (值) 值通常大写字母表示
